```bash
# Create a new script file

```

### Step 4: Start AI Pair Programming
Open your script in your editor and start a conversation with your AI assistant:

**Good first question:**
> "I'm learning shell scripting. I need to write a script that displays 'Hello, [Name]!' with the current date. How do I get started?"

### Step 5: Build Incrementally
1. Start with the shebang: `#!/bin/bash`
2. Add a variable for your name
3. Add the date command
4. Test as you go: `./my-hello-world.sh`

### Step 6: Compare and Learn
After you've written your solution:
```bash
# Compare with reference solution
cat solutions/01-basics/exercise-01-hello-world.sh
```

## ðŸ’¡ Pro Tips

1. **Don't skip the basics** - They're the foundation
2. **Type code yourself** - Don't just copy-paste
3. **Ask "why"** - Understand, don't just make it work
4. **Test edge cases** - What if input is empty? Invalid?
5. **Read error messages** - They tell you what's wrong
