# Resources Directory

This directory contains sample data, configurations, and test files for exercises.

## Structure

- `sample-logs/` - Sample log files for log analysis exercises
- `sample-configs/` - Sample configuration files
- `test-data/` - Test data files

## Usage

These resources are provided to help you practice with realistic data. Feel free to modify them or create your own test data.

## Creating Your Own Test Data

When practicing, you can:
1. Use the provided sample data
2. Generate your own test data
3. Use real data (be careful with sensitive information!)

## Notes

- Sample data is for practice only
- Don't use real credentials or sensitive data
- Feel free to expand these resources as needed
